# Obsidian Callout Syntax

## Basic Format

```markdown
> [!note]
> This is a note callout.
```

## All Callout Types

```markdown
> [!note]
> Standard note

> [!abstract]
> Summary or abstract

> [!info]
> Information

> [!todo]
> To-do item

> [!tip]
> Helpful tip

> [!success]
> Success message

> [!question]
> Question

> [!warning]
> Warning

> [!failure]
> Error or failure

> [!danger]
> Danger alert

> [!bug]
> Bug report

> [!example]
> Example

> [!quote]
> Quotation
```

## Customization

**Custom Title:**

```markdown
> [!tip] Pro Move
> This overrides the default "Tip" title
```

**Collapsible (default open):**

```markdown
> [!note]- Click to collapse
> Content here
```

**Collapsible (default closed):**

```markdown
> [!note]+ Click to expand
> Content here
```

**No Title:**

```markdown
> [!note]-
> Just the content, no title bar
```
